# ⌨️ On Command

> *Master your Mac shortcuts — one keystroke at a time.*

On Command is a macOS keyboard shortcut trainer built with SwiftUI. Practice common Mac shortcuts under pressure — the timer gets faster the longer you survive.

![Platform](https://img.shields.io/badge/platform-macOS-black) ![Swift](https://img.shields.io/badge/Swift-6.2-black) ![SwiftUI](https://img.shields.io/badge/SwiftUI-native-black)

---

## 🎮 How to Play

1. An animation is shown on screen.
2. Press the correct key combination before the timer runs out.
3. A correct answer freezes the timer, plays the animation, then advances to the next challenge.
4. A wrong answer ends the game immediately.
6. Challenges are randomised each session and loop endlessly — the only way out is death.

---

## ⌨️ Shortcuts Covered

| Shortcut | Action |
|----------|--------|
| ⌘A | Select All |
| ⌘B | Bold |
| ⌘C | Copy |
| ⌘D | Duplicate |
| ⌘V | Paste |
| ⌘Z | Undo |
